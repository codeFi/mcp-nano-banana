# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-01-06
### Added
- New `generate_favicons` tool to build complete favicon packages (PNG sizes, favicon.ico, web manifest, optional README, optional ZIP).
- Favicon generation helpers for resizing, ICO creation, manifest/README creation, and ZIP bundling.
- Documentation for favicon generation and local MCP usage examples.

### Changed
- Added dependencies required for favicon packaging (`sharp`, `png-to-ico`, `jszip`).
